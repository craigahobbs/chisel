Chisel Specification Language¶

class chisel.SpecParser(spec=None)¶

The parser class for the Chisel specification language. Parsing can occur at initialization time or by calling the parse_string() method, which can be called repeatedly.

Parameters

spec (str) – An optional specification string to parse

Raises

SpecParserError – A parsing error occurred

actions¶

The dictionary of action name to ActionModel

errors¶

The list of error strings encountered during parsing

finalize()¶

Finalize a parsing operation. You only need to call this method if you set parse() “finalize” argument to False.

Raises

SpecParserError – If type name resolution fails

load(path, spec_ext='.chsl', finalize=True)¶

Recursively load and parse specification files. This method can be called repeatedly.

Parameters

  • path (str) – Directory or file path from which to load and parse specification files

  • spec_ext (str) – The specification file extension. The default specification file extension is “.chsl”.

  • finalize (bool) – If True, resolve names after parsing. Set this argument to False when bulk-parsing related specification files. Be sure to call finalize() when finished.

Raises

SpecParserError – A parsing error occurred

parse(lines, filename='', finalize=True)¶

Parse a specification from an iterator of line strings (e.g an input stream). This method can be called repeatedly.

Parameters

  • lines (Iterable(str)) – An iterator of specification line strings

  • filename (str) – The name of file being parsed (for error messages)

  • finalize (bool) – If True, resolve names after parsing. Set this argument to False when bulk-parsing related specification files. Be sure to call finalize() when finished.

Raises

SpecParserError – A parsing error occurred

parse_string(spec, filename='', finalize=True)¶

Parse a specification string. This method can be called repeatedly.

Parameters

  • spec (str) – The specification string

  • filename (str) – The name of file being parsed (for error messages)

  • finalize (bool) – If True, resolve names after parsing. Set this argument to False when bulk-parsing related specification files. Be sure to call finalize() when finished.

Raises

SpecParserError – A parsing error occurred

types¶

The dictionary of type name to type object
