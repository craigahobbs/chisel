HTML & SVG¶

>>> from chisel import Element
>>> root = Element('html', children=[
...     Element('head', children=[
...         Element('meta', closed=False, charset='UTF-8'),
...         Element('title', inline=True, children=Element('The Title', text=True)),
...         Element('style', _type='text/css', children=Element('a { color: #004B91; }', text_raw=True))
...     ]),
...     Element('body', _class='chsl-index-body', children=[
...         Element('h1', inline=True, children=Element('The Title', text=True))
...     ])
... ])
>>> print(root.serialize())
<!doctype html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>The Title</title>
    <style type="text/css">
a { color: #004B91; }
    </style>
  </head>
  <body class="chsl-index-body">
    <h1>The Title</h1>
  </body>
</html>
class chisel.Element(name, text=False, text_raw=False, closed=True, inline=False, children=None, **attrs)¶

A structured markup language document model element.

Parameters

  • name (str) – The element name.

  • text (bool) – If True, the element name is the text of a text element.

  • text_raw (bool) – If True, the element name is the raw text of a text element.

  • closed (bool) – If True, the element is “closed”.

  • inline (bool) – If True, the element and its children serialized in-line.

  • children (list or Element) – A list of child Element objects (or list objects or None), or Element objects, or None.

  • attrs (dict) – The element’s attributes. For built-ins (e.g. 'class'), prefix the attribute name with an underscore (e.g. '_class'). Key/value pairs with a value of None are ignored.

serialize(indent=' ', html=True)¶

Serialize an element and its children.

Parameters
