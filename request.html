Built-in Requests¶

chisel.create_doc_requests(requests=None, root_path='/doc', api=True, app=True, css=True, cache=True)¶

Yield a series of requests for use with add_requests() comprising the Chisel documentation application. By default, the documenation application is hosted at “/doc/”.

Parameters

  • requests (list(Request)) – A list of requests or None to use the application’s requests

  • root_path (str) – The documentation application URL root path. The default is “/doc”.

  • api (bool) – If True, include the documentation APIs. Two documentation APIs are added, “/doc/doc_index” and “/doc/doc_request”.

  • app (bool) – If True, include the documentation client application.

  • css (bool) – If True, and “app” is True, include “/doc/doc.css” and “/doc/doc.svg”. If you exclude CSS, you can add your own versions of these requests to customize the documentation styling.

  • cache (bool) – If True, cache static request content.

Returns
